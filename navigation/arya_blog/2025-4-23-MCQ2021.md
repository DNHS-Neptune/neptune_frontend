# Q26 Algorithm not suited for parallel computing

The correct answer is A: A parallel computing solution may not be appropriate for an algorithm in which each step requires the output from the preceding step. This is because parallel computing works best when tasks can run independently; if each step depends on the result of the previous one, the tasks must run sequentially, which defeats the purpose of parallelization.

# Q30 How data is transmitted on the Internet

The correct answer is B: Data is broken into packets, which can be sent along different paths. This is correct because on the Internet, data is divided into smaller chunks called packets. Each packet can travel independently across the network, possibly taking different routes to reach the destination. Once all packets arrive, they are reassembled into the original message. This method increases efficiency and reliability, especially if parts of the network are busy or down.
